# Wanderlore - Travel Deeper

A modern travel website that connects travelers with authentic cultural experiences and local storytellers.

## 🌟 Features

### Core Functionality
- **Responsive Design** - Beautiful, mobile-first interface
- **Theme System** - Multiple color themes including Black & White mode
- **User Authentication** - Secure registration and login system
- **User Dashboard** - Personal sidebar with travel management features

### Travel Experience
- **Curated Paths** - Hand-picked travel itineraries with cultural immersion
- **Travel Journal** - Latest stories from global explorers
- **About Section** - Meet our team of cultural storytellers
- **Interactive Navigation** - Smooth scrolling and intuitive user experience

### User Features
- **Mandatory Registration** - First-time users guided through account creation
- **Personal Dashboard** - Left sidebar with:
  - My Trips & Saved Paths
  - Travel Journal & Photos
  - Recommendations & Local Guides
  - Profile Settings & Preferences
  - User Statistics

## 🎨 Themes Available
- Dark Cyber (Default)
- Light Minimal
- Nature Green
- Sunset Warm
- Ocean Blue
- Custom Palette
- **Black & White** - High contrast, minimalist theme

## 🛠️ Technologies Used
- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS Grid and Flexbox
- **JavaScript (ES6+)** - Interactive functionality and theme management
- **Local Storage** - User session and preference management

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/[your-username]/wanderlore.git
   ```

2. Navigate to the project directory:
   ```bash
   cd wanderlore
   ```

3. Open with a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   
   # Using Live Server extension in VS Code
   ```

4. Open your browser and visit `http://localhost:8000`
